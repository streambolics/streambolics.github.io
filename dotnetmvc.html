Le but de ce tutoriel est multiple. D'une part je voudrais évidemment communiquer ce que j'ai appris dans l'écriture d'une application MVC, mais je voudrais surtout éviter de tomber dans le piège de tous les autres tutoriels que j'ai rencontrés jusqu'à présent. En gros, surtout ne pas dire "Voyez comme c'est facile il suffit de cliquer sur ce bouton" et plutôt expliquer le pourquoi du comment de chaque manipulation. # Initialisation d'un projet Commencer dans un répertoire vide. Attention, le nom du répertoire est important, car il va servir de base pour le namespace de l'application qu'on construit. Il est donc important qu'il soit correctement capitalisé. On initialise la source par ```bash dotnet new mvc ``` Profitons-en pour créer un `.gitignore` et lui insérer les deux lignes suivantes correspondant aux répertoires que dotnet utilise pour stocker les binaires et autres artefacts de compilation : ``` /bin/ /obj/ ``` Ce qui nous évitera des surprises par la suite. # Voilà c'est tout Pour le moment...
